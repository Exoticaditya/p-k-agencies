# P.K. Agencies Website

A professional website for P.K. Agencies - Manufacturers & Wholesale Supplier of wedding and event furniture.

## Features

- **Responsive Design**: Works perfectly on all devices (desktop, tablet, mobile)
- **Hero Slider**: Eye-catching image slider on homepage
- **Product Showcase**: Display of all products with beautiful cards
- **Gallery Section**: Filterable gallery to showcase your work
- **Contact Form**: Easy way for customers to reach you
- **WhatsApp Integration**: Direct WhatsApp chat button
- **Smooth Animations**: Professional animations and transitions

## Sections

1. **Home**: Hero slider with call-to-action buttons
2. **About Us**: Company information and background
3. **Products**: Complete product catalog including:
   - Shamiana

4. **Gallery**: Filterable image gallery with categories:
   - All
   - Furniture
   - Chairs
   - Decoration
   - Sofas

5. **Contact**: Contact information and form

## How to Use

1. **Add Your Images**:
   - Place product images in `assets/product/` folder
   - Place gallery images in `assets/gallary/` folder
   - Update image filenames in HTML to match your files

2. **Customize Content**:
   - Edit `index.html` to update text content
   - Modify contact information
   - Add or remove products as needed

3. **Colors**:
   - Primary color: Maroon (#8B0000)
   - Secondary color: Gold (#FFD700)
   - Edit in `styles.css` under `:root` section
